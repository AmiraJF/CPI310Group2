Specifications

The goal of the project is to bring students together while in online class

List of features:

  1. Login/Signup Page
  2. Class list and signup for classes
  3. Study group signup with seperate forum for those groups
  4. Forum for questions regarding classes

We plan to use MySQL for a database because its free and has a lot of information regarding it online so we are less likely to get stuck along the way.
